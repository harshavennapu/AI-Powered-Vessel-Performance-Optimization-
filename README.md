# AI Powered Vessel Performance Optimization Dashboard

## Overview

This project is a maritime analytics dashboard developed using Python and Streamlit. The system processes vessel noon report Excel files and extracts operational KPIs for voyage performance monitoring.

## Features

* Noon Report Excel Upload
* Vessel KPI Extraction
* Speed Performance Analysis
* Fuel Consumption Monitoring
* Weather Condition Alerts
* Interactive Charts
* KPI Report Download

## Technologies Used

* Python
* Streamlit
* Pandas
* Plotly
* OpenPyXL

## Workflow

1. Upload vessel noon report
2. Extract voyage KPIs
3. Analyze vessel performance
4. Monitor weather conditions
5. Generate operational insights
6. Download KPI reports

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```
